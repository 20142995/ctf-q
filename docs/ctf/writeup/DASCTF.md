```

GOOOOxxxx#xxxOOxOOOOOOxOOxxxxOOxxOOOOOxOOxOOOOxxxOOxxOOxxOOOOOxOOxOOOOOOxxxxOOOOOOOOOOOOOOOOOOOOOOOO
```

```
GOOOOxxxx#
xxxOOxOOOO
OOxOOxxxxO
OxxOOOOOxO
OxOOOOxxxO
OxxOOxxOOO
OOxOOxOOOO
OOxxxxOOOO
OOOOOOOOOO
OOOOOOOOOO
```

`jkkjjhjjkjjkkkuukukkuuhhhuukkkk`
